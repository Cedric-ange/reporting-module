# 🚀 Comment Lancer l'Application Maintenant

## Étape 1 : Installation des Dépendances

### Backend
Exécutez ce fichier :
```
C:\Users\angec\reporting-module\install-backend-dep.bat
```

Ou manuellement dans PowerShell :
```powershell
cd C:\Users\angec\reporting-module\backend
npm install express cors multer xlsx body-parser better-sqlite3
```

### Frontend
Exécutez ce fichier :
```
C:\Users\angec\reporting-module\install-frontend-dep.bat
```

Ou manuellement :
```powershell
cd C:\Users\angec\reporting-module\frontend
npm install react react-dom react-router-dom react-scripts axios xlsx date-fns @mui/material @mui/icons-material @emotion/react @emotion/styled
```

## Étape 2 : Lancement de l'Application

### Option Automatisée (Recommandée)
Exécutez :
```
C:\Users\angec\reporting-module\start-complete.bat
```

Ce script lancera automatiquement le backend et le frontend.

### Option Manuel

**Terminal 1 - Backend :**
```bash
cd C:\Users\angec\reporting-module\backend
npm run dev
```

**Terminal 2 - Frontend :**
```bash
cd C:\Users\angec\reporting-module\frontend
npm start
```

## Étape 3 : Accès à l'Application

Une fois lancée :
- **Backend API** : http://
- **Application React** : http://localhost:3000
